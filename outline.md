
# ✅ **Weekly Teaching Plan — WordPress.com (Free Plan)**

**ระยะเวลา:** 10 ครั้ง × 2 ชั่วโมง
**ลักษณะรายวิชา:** ปฏิบัติ (Practice-based)
**รูปแบบกิจกรรม:** Hands-on Learning, Project-based Learning

---

# ✅ **สัปดาห์ที่ 1 — แนะนำ CMS และระบบ WordPress.com**

### **วัตถุประสงค์**

* นักศึกษาสามารถสร้างบัญชี WordPress.com และตั้งค่าเว็บไซต์พื้นฐานได้
* เข้าใจโครงสร้างระบบ CMS ขั้นต้น

### **เนื้อหา**

* ความหมายและประเภทของ CMS
* องค์ประกอบ Frontend / Backend
* Logged-in dashboard interface ของ WordPress.com

### **กิจกรรมการเรียนรู้**

* อาจารย์สาธิตการสมัครบัญชี WordPress.com
* นักศึกษาสร้างเว็บไซต์ใหม่
* ตั้งค่า Site Title, Tagline, Timezone

### **ใบงาน/ภารกิจ**

* สร้างเว็บไซต์ตัวอย่างพร้อมตั้งค่าพื้นฐาน

---

# ✅ **สัปดาห์ที่ 2 — สำรวจ Dashboard และการตั้งค่าระบบ**

### **วัตถุประสงค์**

* นักศึกษาสามารถปรับค่าการแสดงผลของเว็บไซต์ได้ถูกต้อง
* เข้าใจบทบาทผู้ใช้ (User Roles)

### **เนื้อหา**

* เมนู Settings: General, Writing, Discussion
* การเปลี่ยนสถานะเว็บไซต์: Public / Private

### **กิจกรรมการเรียนรู้**

* อาจารย์สาธิตการตั้งค่า URL และเมนูต่าง ๆ
* นักศึกษาปฏิบัติการปรับค่าในเมนู Settings

### **ใบงาน/ภารกิจ**

* ตรวจสอบและบันทึกภาพหน้าจอ Dashboard พร้อมคำอธิบาย

---

# ✅ **สัปดาห์ที่ 3 — ออกแบบโครงสร้างเว็บไซต์ (Information Architecture)**

### **วัตถุประสงค์**

* นักศึกษาสามารถสร้าง Pages/Posts และจัดหมวดหมู่ได้อย่างถูกต้อง
* วางโครงสร้างเนื้อหาแบบเป็นระบบ

### **เนื้อหา**

* Information Architecture
* ความแตกต่างของ Pages vs Posts
* Category และ Tags

### **กิจกรรมการเรียนรู้**

* อาจารย์สาธิตการสร้าง Pages
* นักศึกษาสร้างเพจ: Home, About, Services

### **ใบงาน/ภารกิจ**

* ส่งแผนผังโครงสร้างเว็บไซต์ (Site Map Diagram)

---

# ✅ **สัปดาห์ที่ 4 — การเลือกและปรับแต่งธีมใน WordPress ฟรี**

### **วัตถุประสงค์**

* นักศึกษาสามารถเลือกรูปแบบธีมที่เหมาะสมกับโจทย์
* ปรับแต่ง Header/Footer ได้ในระดับที่ระบบอนุญาต

### **เนื้อหา**

* หลักการเลือกธีม (Theme Selection Strategy)
* Customizer (Appearance)

### **กิจกรรมการเรียนรู้**

* สาธิตการเลือกธีมจาก WordPress Theme Library
* นักศึกษาปรับสี ฟอนต์ Header

### **ใบงาน/ภารกิจ**

* ถ่ายภาพเว็บก่อน/หลังการปรับแต่ง

---

# ✅ **สัปดาห์ที่ 5 — การจัดการเนื้อหาและมัลติมีเดีย**

### **วัตถุประสงค์**

* นักศึกษาสามารถใช้ Block Editor ในการสร้างเนื้อหาอย่างถูกต้อง
* ใช้รูปภาพ/วิดีโอได้โดยไม่ละเมิดลิขสิทธิ์

### **เนื้อหา**

* Gutenberg Blocks
* การแทรก Media
* Creative Commons / Free Image Sources

### **กิจกรรมการเรียนรู้**

* อาจารย์แสดงการใช้ Block ประเภท Text/Media
* นักศึกษาเขียนบทความพร้อมรูปภาพประกอบ

### **ใบงาน/ภารกิจ**

* สร้างหน้า "About Us" อย่างเป็นทางการ

---

# ✅ **สัปดาห์ที่ 6 — การสร้างเมนูและระบบนำทางเว็บไซต์**

### **วัตถุประสงค์**

* สามารถสร้าง Navigation Menu ที่เป็นระบบและใช้งานได้จริง

### **เนื้อหา**

* Organization of Navigation
* การเพิ่มเมนูหลักและเมนูย่อย

### **กิจกรรมการเรียนรู้**

* อาจารย์สาธิตการสร้างเมนู
* นักศึกษากำหนดเมนูหลัก 1 ชุด

### **ใบงาน/ภารกิจ**

* ส่งโครงสร้างเมนูในรูปแบบภาพและคำอธิบาย

---

# ✅ **สัปดาห์ที่ 7 — การจัดการสื่อและการเพิ่มประสิทธิภาพเว็บไซต์**

### **วัตถุประสงค์**

* นักศึกษาสามารถจัดการไฟล์สื่อใน Media Library และบีบอัดรูปภาพได้

### **เนื้อหา**

* ข้อจำกัดพื้นที่สื่อใน WordPress ฟรี
* การลดขนาดภาพเพื่อเพิ่มความเร็ว

### **กิจกรรมการเรียนรู้**

* สาธิตการบีบอัดรูปภาพ
* นักศึกษาอัปโหลดและจัดหมวดหมู่ไฟล์สื่อ

### **ใบงาน/ภารกิจ**

* รายงานขนาดไฟล์ก่อน/หลังการบีบอัด

---

# ✅ **สัปดาห์ที่ 8 — ออกแบบหน้าเว็บด้วย Block Patterns และ Layouts**

### **วัตถุประสงค์**

* ออกแบบหน้าเว็บโดยใช้ Block Patterns

### **เนื้อหา**

* Layout Design Basics
* Grid System / Visual Hierarchy
* Block Patterns

### **กิจกรรมการเรียนรู้**

* อาจารย์สาธิตการใช้ Patterns
* นักศึกษาสร้างหน้า Landing Page

### **ใบงาน/ภารกิจ**

* ส่งหน้า Landing Page พร้อมคำอธิบายส่วนประกอบ

---

# ✅ **สัปดาห์ที่ 9 — การตั้งค่าความเป็นส่วนตัวและตรวจสอบการแสดงผล**

### **วัตถุประสงค์**

* นักศึกษาสามารถตั้งค่า Static Homepage และตรวจสอบการแสดงผล Mobile

### **เนื้อหา**

* Privacy Settings
* Mobile-First Design
* Responsive Testing

### **กิจกรรมการเรียนรู้**

* สาธิตการตั้ง Static Page
* นักศึกษาทดสอบบนหลายอุปกรณ์

### **ใบงาน/ภารกิจ**

* Screenshot การแสดงผลบนมือถือ และวิเคราะห์ปัญหา

---

# ✅ **สัปดาห์ที่ 10 — การประเมินผลงานและการนำเสนอเว็บไซต์**

### **วัตถุประสงค์**

* นำเสนอเว็บไซต์ที่พัฒนาและวิเคราะห์คุณภาพเชิงวิชาการได้

### **เนื้อหา**

* Web Evaluation Criteria
* UX/UI Assessment
* Self-reflection

### **กิจกรรมการเรียนรู้**

* นักศึกษานำเสนอผลงานเว็บไซต์ต่อชั้นเรียน
* peer review ตาม rubric

### **ใบงาน/ภารกิจ**

* เขียน Reflection 1 หน้า: “สิ่งที่ได้เรียนรู้จากการพัฒนาเว็บไซต์ด้วย WordPress ฟรี”
