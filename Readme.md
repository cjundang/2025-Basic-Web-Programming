# INF67-175 Web Development

## หน่วยที่ 1 : แนะนำ CMS และระบบ WordPress.com (สัปดาห์ที่ 1) 
[อ่านเพิ่มเติม](/contents/chapter-01.md)


ระบบจัดการเนื้อหา (Content Management System: CMS) คือซอฟต์แวร์สำหรับสร้าง จัดการ และเผยแพร่เนื้อหาบนเว็บไซต์โดยไม่จำเป็นต้องเขียนโค้ด โดยมีแนวคิดสำคัญคือการแยก “เนื้อหา” ออกจาก “การออกแบบ” เนื้อหาถูกเก็บในฐานข้อมูล ขณะที่โครงสร้างและหน้าตาเว็บไซต์กำหนดด้วยเทมเพลตหรือธีม เมื่อมีผู้เข้าชม CMS จะดึงเนื้อหามาผสานกับเทมเพลตและแสดงผลเป็นหน้าเว็บแบบไดนามิก

ประโยชน์หลักของ CMS ได้แก่ การลดเวลาและค่าใช้จ่ายในการพัฒนาเว็บไซต์ ทำให้ผู้ใช้ทั่วไปสามารถดูแลเว็บไซต์ได้ การสนับสนุนการทำงานร่วมกันผ่านระบบผู้ใช้หลายบทบาท และการปรับแต่งเว็บไซต์ผ่านธีมและปลั๊กอิน เช่น ระบบบทความ แกลเลอรี หรือร้านค้าออนไลน์ ตัวอย่างแพลตฟอร์มที่สำคัญ ได้แก่ WordPress, Shopify, Joomla, Drupal, Wix, Squarespace, HubSpot CMS, BigCommerce และ Mambo

อย่างไรก็ตาม CMS มีข้อจำกัดด้านเทคนิค เช่น ความซับซ้อนเมื่อออกแบบธีมหรือปลั๊กอินเอง และเส้นโค้งการเรียนรู้ของแต่ละระบบ ผู้ใช้จึงควรพิจารณาความต้องการ ทักษะ งบประมาณ และความยืดหยุ่นในอนาคตในการเลือกใช้ CMS ให้เหมาะสมกับบริบทของตนเอง

## หน่วยที่ 2 : Dashboard และการตั้งค่าระบบ (สัปดาห์ที่ 2)

* ภาพรวมโครงสร้างเมนู Dashboard
* การตั้งค่าในเมนู Settings

  * General
  * Writing
  * Discussion
* การกำหนดสถานะเว็บไซต์

  * Public
  * Private
* แนวคิดและบทบาทของผู้ใช้ (User Roles) ในระบบ

## หน่วยที่ 3 : โครงสร้างเนื้อหาและ Information Architecture (สัปดาห์ที่ 3)

* แนวคิด Information Architecture (IA)
* ความแตกต่างระหว่าง

  * Pages
  * Posts
* การจัดหมวดหมู่เนื้อหา

  * Categories
  * Tags
* การออกแบบโครงสร้างเว็บไซต์ (Site Structure / Site Map)

---

## หน่วยที่ 4 : การเลือกและปรับแต่งธีมใน WordPress (สัปดาห์ที่ 4)

* หลักการเลือกธีมให้เหมาะสมกับวัตถุประสงค์ของเว็บไซต์
* การใช้ระบบปรับแต่งหน้าตาเว็บไซต์ (Customizer / Appearance)
* การปรับแต่งองค์ประกอบพื้นฐาน

  * Header
  * Footer
  * สี (Color Scheme)
  * ฟอนต์ (Typography)

---

## หน่วยที่ 5 : การจัดการเนื้อหาและมัลติมีเดีย (สัปดาห์ที่ 5)

* การใช้ Block Editor (Gutenberg)

  * Text Blocks
  * Media Blocks
* การแทรกและจัดการสื่อ

  * รูปภาพ
  * วิดีโอ
* หลักการใช้สื่ออย่างถูกต้องตามลิขสิทธิ์

  * Creative Commons
  * แหล่งรูปภาพฟรี (Free Image Sources)
* การสร้างหน้า/บทความเชิงทางการ (เช่น About Us / บทเรียน)

## หน่วยที่ 6 : การสร้างเมนูและระบบนำทาง (Navigation) (สัปดาห์ที่ 6)

* หลักการออกแบบระบบนำทาง (Organization of Navigation)
* การสร้างและจัดการเมนูหลัก (Main Menu)

* ความสัมพันธ์ระหว่างเมนูกับโครงสร้างเนื้อหา

## หน่วยที่ 7 : การจัดการสื่อและการเพิ่มประสิทธิภาพเว็บไซต์ (สัปดาห์ที่ 7)

* ข้อจำกัดด้านพื้นที่จัดเก็บสื่อใน WordPress.com (Free Plan)
* การจัดการ Media Library

  * การอัปโหลด
  * การจัดหมวดหมู่/โฟลเดอร์ (ถ้ามี)
* เทคนิคการลดขนาดไฟล์ภาพ

  * การบีบอัดภาพ (Image Compression)
  * ผลของขนาดไฟล์ต่อความเร็วเว็บไซต์

## หน่วยที่ 10 : การประเมินผลงานเว็บไซต์และการสะท้อนการเรียนรู้ (สัปดาห์ที่ 10)
